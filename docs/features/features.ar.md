## General

### المظهر المعتِم

يتيح لك الاختيار بين إصدار داكن مخصص ووإصدار كلاسيكيأساسي في وحدة التحكم بالمجتمع.

*** promo
_آلي:_ will use the theme defined in the system settings.  
_يدوي:_ will add a button to the Community Console that lets you switch the theme.
***

_وحدة التّحكُّم بالمجتمع_ | [\[demo\]](https://raw.githubusercontent.com/avm99963/infinitegforums/HEAD/src/features/ccDarkTheme/presentation/options/assets/screenshot.avif)

### إعادة التّوجيه إلى وحدة التّحكم بالمجتمع

إعادة توجيه جميع المواضيع المفتوحة في الواجهة العامّة للمنتدى إلى وحدة التحكم بالمجتمع.

_Tailwind Basic_

### الوضع المُدمَج

تقليل المسافة البيضاء على واجهة المستخدم.

_Tailwind Basic, وحدة التّحكُّم بالمجتمع_ | [\[demo\]](https://raw.githubusercontent.com/avm99963/infinitegforums/HEAD/src/features/uiSpacing/presentation/options/assets/screenshot.avif)

Best-effort workaround for the issues discussed at [pekb/381989895](https://support.google.com/s/community/forum/51488989/thread/381989895).

## Message composer

### Fix link dialog

Patches the bug that incorrectly opens the link dialog multiple times when adding or editing a link.

_وحدة التّحكُّم بالمجتمع_

### Minor UI enhancements

#### Fix canned responses popup

Fixes the selection popup that appears when inserting a CR, so it is shown correctly when some CR has a very long title.

## Old message composer

*** promo
These features only affect the Community Console's old message composer, which is shown when creating a new thread or canned response, when pressing `r` inside a thread, or when creating a new reply in the old thread view. The new message composer doesn't suffer from these issues.
***

### إصلاح روابط السَّحب والإفلات

السَّماح بسحب وإفلات الروابط إلى محرر النُّصوص مع الحفاظ على نصِّ الرَّابط.

_وحدة التّحكُّم بالمجتمع_

### منع حفظ المسودَّات

امنع حفظ مسودَّات ردودك أثناء الكتابة على خوادم جوجل في وحدة التحكم بالمجتمع.

_وحدة التّحكُّم بالمجتمع_

### تحميل مسودات الرسائل عند الرد

Enables the `enableLoadingDraftMessages` Community Console flag, which enables recovering an existing draft saved in the Google servers when you start a new reply.

_وحدة التّحكُّم بالمجتمع_
